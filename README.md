# Cloud Infrastructure & Security Monitoring Lab

## Overview
This project is a **local AWS-like environment** for learning cloud infrastructure, IAM, security groups, EC2 instances, and security monitoring.  
It uses **Docker**, **LocalStack**, and **Terraform** to simulate AWS services locally without touching real AWS.  


## Tech Stack
- **Docker** — runs the lab containers  
- **LocalStack** — simulates AWS APIs locally  
- **Terraform** — provisions infrastructure as code  
- **AWS CLI / awslocal** — interacts with the fake AWS  
- **Bash scripts** — generate logs and simulate security events
